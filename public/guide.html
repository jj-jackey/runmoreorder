🎯 시스템 개요

이 시스템은 주문서 Excel 파일을 발주서 형태로 자동 변환하여 업무 효율성을 극대화하는 도구입니다.

💡 핵심 가치
• 수작업 발주서 작성 시간을 90% 단축
• 사람의 실수로 인한 오류 최소화
• 일관된 형태의 발주서 생성
• 이메일 자동 발송으로 즉시 전달

⚡ 주요 기능

📁 다양한 파일 지원

Excel (.xlsx, .xls), CSV 파일 모두 지원하며, 대용량 파일도 안정적으로 처리합니다.

🤖 AI 자동 매칭

OpenAI를 활용하여 필드명을 분석해 자동으로 최적의 매칭을 제공합니다.

📋 템플릿 시스템

자주 사용하는 매칭 규칙을 템플릿으로 저장하여 재사용할 수 있습니다.

📧 자동 이메일 발송

발주서 생성 후 즉시 이메일로 전송하거나 예약 발송도 가능합니다.

🔍 실시간 미리보기

매칭 결과를 실시간으로 확인하고 수정할 수 있습니다.

📊 데이터 검증

이메일, 전화번호, 날짜 형식을 자동으로 검증해 오류를 사전에 방지합니다.

🔄 기본 사용법

작업 모드 선택

• 파일 업로드: 주문서와 발주서 템플릿 모두 업로드

• 직접 입력: 주문서만 업로드하고 기본 템플릿 사용

• 기본 템플릿: 시스템 제공 템플릿 사용

• 저장된 템플릿: 이전에 저장한 템플릿 재사용

파일 업로드 및 분석

• 드래그 앤 드롭 또는 파일 선택으로 업로드

• 시스템이 자동으로 파일 구조 분석

• 헤더와 데이터 행을 자동 감지

필드 매칭 설정

• AI 자동 매칭 또는 수동 드래그 앤 드롭

• 실시간 미리보기로 결과 확인

• 필요시 수동으로 세부 조정

발주서 생성 및 확인

• 매칭 규칙 적용하여 자동 변환

• 생성된 발주서 다운로드

• 필요시 이메일로 즉시 전송

🔗 필드 매칭

매칭 유형

1. 자동 매칭: AI가 컬럼명과 데이터 분석하여 추천
2. 수동 매칭: 사용자가 직접 드롭다운에서 선택
3. 직접 입력: 고정값 또는 수식 입력

매칭 우선순위

1. 직접 입력값 (최고 우선순위) 2. 매핑된 주문서 컬럼 데이터 3. 빈 값 (기본값)

🤖 AI 자동 매칭 설정

API 키 발급

1. OpenAI Platform 접속

2. 계정 로그인 또는 회원가입

3. "Create new secret key" 클릭

4. API 키 복사 (sk-로 시작하는 긴 문자열)

시스템에 API 키 등록

1. 우상단 🔧 API 키 설정 버튼 클릭

2. API 키 입력란에 복사한 키 붙여넣기

3. 🚀 시작하기 버튼 클릭

4. 검증 완료 후 메인 페이지로 자동 이동

AI 기능 사용

• 우상단이 🤖 AI 기능 사용 가능으로 변경됨

• 🤖 AI 자동 매핑 버튼 활성화

• 더 정확하고 빠른 매핑 가능

📋 템플릿 시스템

템플릿 저장

자주 사용하는 매칭 규칙을 템플릿으로 저장하여 다음에 재사용할 수 있습니다.

빠른 템플릿 검색

🔍 새로운 기능!
저장된 템플릿 사용 모드에서 템플릿 이름을 입력하여 빠르게 검색하고 선택할 수 있습니다.

📧 이메일 시스템

즉시 발송

발주서 생성 후 바로 이메일로 전송할 수 있습니다.

예약 발송

원하는 날짜와 시간을 설정하여 예약 발송이 가능합니다.

발송 이력 관리

모든 이메일 발송 이력을 확인하고 관리할 수 있습니다.

💡 활용 팁

💡 성능 최적화
• 1000행 이하: 최적 성능
• 5000행 이하: 양호한 성능
• 10000행 이상: 파일 분할 권장

자주 사용하는 기능

  • 템플릿 활용: 반복 작업 시 시간 절약
  • 미리보기 확인: 변환 전 결과 검토
  • 데이터 검증: 이메일, 전화번호 형식 확인
  • 일괄 처리: 대량 데이터 효율적 처리

🛠 문제해결

일반적인 문제

❌ 파일 업로드 실패

  • 파일 형식 확인 (.xlsx, .xls, .csv만 지원)
  • 파일 크기 확인 (10MB 이하 권장)
  • 네트워크 연결 상태 확인

🔍 매칭 결과가 이상한 경우

  • 컬럼명과 데이터 형식 확인
  • 수동으로 매칭 조정
  • 미리보기로 결과 검토

📧 이메일 발송 실패

  • 수신자 이메일 주소 확인
  • 네트워크 연결 상태 확인
  • 첨부파일 크기 확인
